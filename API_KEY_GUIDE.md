# 국토교통부 실거래가 API 키 발급 가이드

## 📋 준비사항
- 이메일 주소
- 휴대폰 번호 (본인인증용)

## 🔑 API 키 발급 단계별 가이드

### 1단계: 공공데이터포털 회원가입

1. [공공데이터포털](https://www.data.go.kr) 접속
2. 우측 상단 "회원가입" 클릭
3. 이용약관 동의 → 본인인증 → 회원정보 입력

### 2단계: 로그인 및 API 검색

1. 로그인 후 메인 페이지에서 검색창에 **"국토교통부 아파트 매매 실거래"** 입력
2. 검색 결과에서 **"국토교통부_아파트 매매 실거래가 상세 자료"** 클릭 ⚠️ 중요!
   - 정확한 명칭: 국토교통부_아파트 매매 실거래가 상세 자료
   - 제공기관: 국토교통부
   - **주의**: "상세 자료"를 선택해야 더 많은 정보를 받을 수 있습니다

### 3단계: 활용신청

1. 해당 페이지에서 **"활용신청"** 버튼 클릭
2. 활용신청 정보 입력:
   - **활용목적**: 개인 웹사이트에서 아파트 추천 서비스
   - **활용용도**: 비영리
   - **라이선스 표시**: 동의
3. **"동의 및 활용신청"** 클릭

### 4단계: 승인 및 키 확인

1. 신청 즉시 자동승인 (평일 기준)
   - 주말/공휴일은 다음 영업일에 승인
2. **마이페이지 → 오픈API → 신청현황** 이동
3. 승인된 API 목록에서 **"국토교통부_아파트 매매 실거래가 자료"** 클릭
4. **서비스키** 확인 (2가지 제공):
   - **Encoding 키** (URL 인코딩된 키) ✅ 이것을 사용
   - **Decoding 키** (원본 키)

### 5단계: API 키 설정

#### Windows (명령 프롬프트)
```cmd
set MOLIT_API_KEY=여기에_Encoding_키_붙여넣기
```

#### Mac/Linux (터미널)
```bash
export MOLIT_API_KEY="여기에_Encoding_키_붙여넣기"
```

#### 영구 설정 (선택사항)
`.env` 파일 생성:
```
MOLIT_API_KEY=여기에_Encoding_키_붙여넣기
```

## ⚠️ 주의사항

1. **일일 트래픽 제한**: 개발계정은 1,000건/일
2. **키 보안**: API 키를 GitHub에 올리지 마세요
3. **Encoding 키 사용**: 반드시 Encoding 키를 사용하세요
4. **승인 시간**: 평일은 즉시, 주말은 월요일 승인

## 🔍 확인 방법

API 키가 제대로 설정되었는지 확인:
```bash
# Mac/Linux
echo $MOLIT_API_KEY

# Windows
echo %MOLIT_API_KEY%
```

## 💡 자주 묻는 질문

**Q: API 키 발급이 거부되었어요**
A: 활용목적을 구체적으로 작성하세요. 예: "개인 포트폴리오용 부동산 정보 제공 웹서비스"

**Q: Encoding키와 Decoding키 중 뭘 써야 하나요?**
A: Encoding키를 사용하세요. URL에 포함될 때 안전합니다.

**Q: 1,000건 제한을 늘릴 수 있나요?**
A: 운영계정으로 전환 신청하면 최대 10,000건까지 가능합니다.
